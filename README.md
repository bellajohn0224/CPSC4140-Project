REQUIREMENTS
- Node.js v18+ (available on SoC machines by default)
- npm v9+
 
SETUP & RUN
1. Unzip the project and navigate into the folder:
       cd milestone3
 
2. Install dependencies:
       npm install
 
3. Start the development server:
       npm run dev
 
4. Open your browser to http://localhost:5173
   (If that port is in use, Vite will print the correct URL in the terminal.)
 
5. To stop the server, press Ctrl+C.
